# AICam

AI 攝影教練相機（iPhone 原生）— 即時構圖指導、AI 調色、AI 抓拍、AI 篩選、專業手動控制 + RAW。黑白質感 UI。

> **⚠️ 接手開發前必讀 [MASTER-PLAN.md](MASTER-PLAN.md)** — 完整功能表、架構決策（含禁用 ARKit 等硬性決策）、P0–P5 路線圖與執行鐵律都在裡面。

- 平台：iOS 17+（Swift / SwiftUI / AVFoundation / Vision / Core ML / Metal）
- 建置：GitHub Actions（本機無 Xcode；ubuntu 跑 `AICamCore` 測試、macos 產 unsigned IPA 側載）
- 狀態：規劃完成，待 P0 動工
